1. Mi az ERP?

Az ERP az Enterprise Resource Planning Software kifejezést jelenti, és egy integrált számítógépes rendszer, amelyet a vállalat erőforrásainak hatékony kezelésére használunk. Biztosítja a zökkenőmentes információ áramlást egy vállalkozás vagy egy vállalat különféle részlegei között, és kezeli a munkafolyamatokat.

3. Melyek a különböző SAP termékek?

SAP R/3 - Az SAP R/2 utódja és piacvezető az ERP-ben. Az R/3 háromszintű architektúrát jelent, azaz a bemutatás, a logika és az adat szintjét. Számos olyan modullal rendelkezik, mint SD, FI, HR stb., amelyek szinte minden vállalati osztályt lefednek.

mySAP - Ez egy SAP termékcsalád, amely az SAP R/3-n kívül tartalmaz SRM, PLM, CRM, SCM modulokat.

4. Mi a NetWeaver?

A Netweaver egy olyan integrált technológiai platform, amely lehetővé teszi a mySAP csomag összes termékének futtatását az SAP Web Application Server (SAP WEBA) néven ismert netweaver egyetlen példányán.

A Netweaver használatának előnye, hogy az SAP-adatokhoz interneten (http protokoll) vagy akár mobilon is hozzáférhetünk.

6. Az SAP egy adatbázis?

NEM. Az SAP nem adatbázis, hanem egy olyan alkalmazás, amely más gyártók, például Oracle, SQL Server stb. által biztosított adatbázisokat használ fel.

7. Mik a változók?

A változók egy olyan lekérdezés paraméterei, amelyeket a paraméter-lekérdezés meghatározásában állítanak be, és nem kerülnek értékekkel kitöltésre, amíg a lekérdezések bekerülnek a munkafüzetekbe.

8. Milyen típusú változók léteznek?

  • Characteristics variable
  • Hierarchies
  • Hierarchy nodes
  • Text
  • Formulas
  • Processing Types
  • Replacement Path
  • User entry/default type

9. Mondja meg, mi a különbség az SAP BASIS és az SAP ABAP között?

Az SAP ABAP az SAP-n belül az testreszabáshoz, az űrlapok létrehozásához, a jelentések készítéséhez stb. használt programozási nyelv. Míg az SAP BASIS az SAP adminisztrációs modulja a kódváltozások, frissítések, az adatbázis adminjának, a hálózati beállításoknak stb. vezérlésére szolgál.

11. Magyarázza el, mi az Extractor!

Az SAP forrásrendszerben az extraktorok adat-visszakeresési mechanizmusok. Ez kitölti az adatforrás kivonat szerkezetét az SAP forrásrendszer adatkészleteinek adataival.
